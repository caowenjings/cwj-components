# cwj-components

一个基于 dumi 搭建的 react 组件库,

## Getting Started

Install dependencies,

```bash
$ npm i cwj-components
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run build
```
